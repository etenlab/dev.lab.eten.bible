ETEN Innovation Lab

The Lab seeks to accelerate Bible translation through advancements in translation resources, quality assurance, artificial intelligence, and crowd sourcing. To facilitate collaboration and reduce rework, the lab is providing a hosted environment for software projects and APIs. If you would like to contribute in any way, request funding for a project, or use the APIs in your applications, please email admin@etenlab.com.

Utility APIs

Single Sign On
In Development
  • SSO service for anyone one to use in any application
  • Simplifies user registration and profile management
App Registration
In Development
  • Registers an org's application in the ecosystem
  • Some APIs require an app to be registered in order to create new rows in their table
Voting
In Development
  • Allows users to vote on any content
  • Can be used for any voting concept an app may need
Site Text Keying
In Development
  • Provides user interface text in all languages available
  • Can be used by applications to make interfaces available in any language
Site Text Translation
In Development
  • Used by translators to translate site text interface text from one language to another
  • Applications can use the Showcase app's translation interface to have their site text translated into any language a translator is available for
Authorization
In Development
  • Validate a user's access to data
Discussion
In Development
  • Allows application developers to add a discussion box on any content
  • Enables users to discuss and share resources
Organization
In Development
  • Management for organization registration
  • Authorization for an org's admins
File
In Development
  • Allows users and applications the ability to upload files
  • Enables access control on each file per organization
Organization Language Index
In Development
  • Enables any organization to manage their own table of languages
  • Entries can reference data from other organizations if permitted
  • Allows organizations to maintain their own language data while still working with other orgs on theirs, with everyone sharing a common view
Translation
In Development
  • Generic translation service for translators to translate content from one language to another
  • Can enable target languages to become a source language
Peer-to-Peer
In Development
  • Only used when running a local server
  • Allows offline teams to share data between teams and team members
  • Users can transfer data from online networks to offline devices
